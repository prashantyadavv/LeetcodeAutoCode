
# LeetCode AutoCode Extension

A Chrome extension that enhances the LeetCode coding experience by automatically generating code templates and streamlining the coding workflow. It helps users start solving problems faster by reducing repetitive setup and improving productivity.

## 🚀 Features

* ⚡ Automatically generates code templates for LeetCode problems.
* 🧠 Detects the current programming language.
* 📋 Reduces repetitive coding boilerplate.
* 🎯 Works directly within the LeetCode editor.
* 💻 Lightweight and easy to use.

## 📂 Project Structure

```text
├── background.js        # Background service worker
├── content.js           # Injects functionality into LeetCode pages
├── manifest.json        # Chrome extension configuration
├── popup.html           # Extension popup UI
├── popup.css            # Popup styling
├── popup.js             # Popup logic
├── icons/               # Extension icons
└── README.md
```

## 🛠️ Installation

1. Clone this repository:

```bash
git clone https://github.com/prashantyadavv/leetcode-extension.git
```

2. Open **Google Chrome**.

3. Navigate to:

```
chrome://extensions/
```

4. Enable **Developer Mode**.

5. Click **Load unpacked**.

6. Select the project folder.

7. Open LeetCode and start solving problems.

## 📖 How It Works

1. Visit any problem on LeetCode.
2. The extension detects the problem page.
3. It automatically prepares the editor with the appropriate code template.
4. Begin coding immediately without manually writing boilerplate.
